# 贡献者公约行为准则

## 我们的承诺

我们作为成员、贡献者和领导者，承诺为我们社区中的每个人创造一个免受骚扰的体验，无论其年龄、体型、可见或不可见残疾、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、种姓、肤色、宗教或性取向和性认同如何。

我们承诺以有助于建立一个开放、热情、多元、包容和健康的社区的方式行事和互动。

## 我们的标准

有助于为我们的社区创造积极环境的行为示例包括：

* 表现出对他人的同情和友善。
* 尊重不同的意见、观点和经验。
* 给予并优雅地接受建设性的反馈。
* 承担责任并为我们的错误向受影响的人道歉，并从中吸取教训。
* 关注对整个社区最有利的事情，而不仅仅是对我们个人而言。

不可接受的行为示例包括：

* 使用性化的语言或图像，以及任何形式的性关注或挑逗。
* 钓鱼、侮辱或贬损性评论，以及人身或政治攻击。
* 公开或私下骚扰。
* 未经他人明确许可，发布他人的私人信息，例如物理或电子邮件地址。
* 在专业环境中可以合理地认为是不适当的其他行为。

## 执行责任

社区领导者负责澄清和执行我们可接受行为的标准，并将采取适当和公正的纠正措施，以回应他们认为不适当、具有威胁性、冒犯性或有害的任何行为。

社区领导者有权力和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、维基编辑、问题和其他贡献，并将在适当时沟通缓和决定的原因。

## 范围

本行为准则适用于所有社区空间，并且当个人在公共场所正式代表社区时也适用。代表我们社区的示例包括使用官方电子邮件地址、通过官方社交媒体帐户发布帖子或在在线或离线活动中担任指定的代表。

## 执行

虐待、骚扰或以其他方式不可接受的行为的实例可以报告给负责执行的社区领导者，邮箱地址为 <mannaandpoem@gmail.com>。所有投诉都将得到及时和公正的审查和调查。

所有社区领导者都有义务尊重任何事件报告者的隐私和安全。

## 执行指南

社区领导者将遵循这些社区影响指南，以确定他们认为违反本行为准则的任何行为的后果：

### 1. 纠正

**社区影响**：在社区中使用不适当的语言或其他被认为不专业或不受欢迎的行为。

**后果**：社区领导者的私下书面警告，明确说明违规行为的性质，并解释为什么该行为不适当。可能会要求公开道歉。

### 2. 警告

**社区影响**：通过单个事件或一系列行动的违规行为。

**后果**：警告，并说明继续行为的后果。在指定的时间段内，不得与相关人员进行互动，包括与执行行为准则的人员进行非主动互动。这包括避免在社区空间以及社交媒体等外部渠道中进行互动。违反这些条款可能会导致暂时或永久禁令。

### 3. 临时禁令

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定的时间段内，禁止与社区进行任何形式的互动或公开交流。在此期间，不允许与相关人员进行任何公开或私下互动，包括与执行行为准则的人员进行非主动互动。违反这些条款可能会导致永久禁令。

### 4. 永久禁令

**社区影响**：表现出违反社区标准的模式，包括持续的不当行为、骚扰个人或攻击或贬低各类个人。

**后果**：永久禁止在社区内进行任何形式的公开互动。

### Slack 和 Discord 礼仪

这些 Slack 和 Discord 礼仪指南旨在为所有社区成员营造一个包容、尊重和高效的环境。通过遵循这些最佳实践，我们确保有效的沟通和协作，同时最大限度地减少中断。让我们共同努力，建立一个支持和欢迎的社区！

* 以尊重和专业的方式进行交流，避免使用讽刺或 harsh 的语言，并记住语气在文本中很难解释。
* 使用主题帖进行特定讨论，以保持频道井井有条且更易于关注。
* 仅当他们的意见至关重要或紧急时才标记其他人，并谨慎使用 @here、@channel 或 @everyone 以最大限度地减少中断。
* 要有耐心，因为开源贡献者和维护者通常有其他承诺，可能需要时间才能做出回应。
* 在最相关的频道中发布问题或讨论（[discord - #general](https://discord.com/channels/1125308739348594758/1138430348557025341)）。
* 在寻求帮助或提出问题时，请包含必要的详细信息，如链接、屏幕截图或清晰的解释，以提供上下文。
* 尽可能在公共频道中进行讨论，以便其他人可以从对话中受益，除非此事敏感或私密。
* 始终遵守[我们的标准](https://github.com/mannaandpoem/OpenManus/blob/main/CODE_OF_CONDUCT.md#our-standards)，以确保一个热情和协作的环境。
* 如果您选择将某个频道静音，请考虑设置您仍然感兴趣的主题的提醒，以保持参与。对于 Slack，转到设置 → 通知 → 我的关键字，以添加在消息中出现时会通知您的特定关键字。例如，如果您来这里是为了讨论 LLM，如果频道太忙，请将其静音，但设置通知以仅在消息中出现“LLM”时提醒您。此外，对于 Discord，请转到频道通知并选择最能描述您需要的选项。
